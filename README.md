# Jean Cocteau International School

Site vitrine premium pour Jean Cocteau International School, école d'excellence de la maternelle au lycée.

## Stack technique

- **React 18** + **Vite** (build ultra-rapide)
- **React Router** (navigation SPA)
- **Tailwind CSS** (design moderne et responsive)
- **Lucide React** (icônes)

## Démarrer le projet

```bash
npm install
npm run dev
```

Le site sera accessible sur `http://localhost:5173`.

## Build production

```bash
npm run build
npm run preview
```

## Pages

- `/` — Accueil (hero, présentation, cycles, atouts, actualités, témoignages, CTA)
- `/a-propos` — Histoire, vision, valeurs, équipe, campus
- `/cycles` — Vue d'ensemble des 4 cycles
  - `/cycles/maternelle` — 3-5 ans
  - `/cycles/primaire` — 6-10 ans
  - `/cycles/college` — 11-14 ans
  - `/cycles/lycee` — 15-18 ans
- `/admissions` — Processus, documents, frais, formulaire
- `/vie-scolaire` — Clubs, événements, services
- `/actualites` — Blog filtrable par catégorie
- `/galerie` — Galerie photos avec lightbox
- `/contact` — Formulaire + Google Maps
- `/faq` — Questions fréquentes par catégorie
- `/carrieres` — Offres d'emploi

## Charte graphique

- **Bleu marine** (`navy-900`) — couleur principale
- **Doré** (`gold-500`) — accents premium
- **Blanc** — fond principal
- **Gris clair** (`navy-50`) — sections alternées
- Typographie : Playfair Display (titres) + Inter (corps)
